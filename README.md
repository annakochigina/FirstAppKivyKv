Приложение "Трекер привычек"
Хранит список привычек, которые нужно отмечать каждый день. Можно добавлять новые привычки, при повторном открытии они сохраняются
Все привычки хранятся в json-файле
При повторном открытии также ранее отмеченные привычки остаются выбранными. Предполагается, что
будет сброс трекеров, когда наступает ночь, но это уже в следующей версии приложения:)
